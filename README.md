# Generador de Contraseñas Seguras

## Descripción
Este proyecto consiste en un sistema que permite generar, evaluar, guardar y visualizar contraseñas.

## Funcionalidades
- Generar contraseña segura
- Evaluar seguridad de contraseña
- Guardar contraseña
- Visualizar contraseñas

## herramientas utilizadas
- Python
- Graphviz

## Estructuras utilizadas
- Variables
- Condicionales (if)
- Bucles (while, for)
- Entrada y salida de datos

## Diagramas
Se incluye el diagrama de flujo general del sistema.
